## 一、贺卡设计的核心要素

### 1. 情感定位（最重要）
- 贺卡是**情感载体**，不是炫技作品
- 先想：这张卡要传递什么情感？
  - 生日 → 欢乐、庆祝、成长
  - 纪念日 → 浪漫、珍惜、承诺
  - 鼓励 → 支持、温暖、力量
  - 节日 → 氛围、仪式感、祝福

### 2. 视觉层次
```
第一层：背景（渐变/图案/粒子）
第二层：装饰元素（emoji/图形/光效）
第三层：卡片主体（边框+内容）
第四层：文字（名字→日期→祝福→署名）
```

### 3. 动画节奏
- **入场动画**：文字依次出现，营造揭晓感
- **背景动画**：粒子/粒子持续流动，不抢焦点
- **交互动画**：点击翻转、悬停反馈
- **呼吸动画**：卡片轻微浮动，增添生命力

## 二、配色理论

### 色轮基础
- **互补色**：180度相对（蓝+橙、紫+黄）
- **邻近色**：30度相邻（红+橙、粉+红）
- **冷暖对比**：暖色（红/橙/黄）= 温暖/喜庆；冷色（蓝/紫/绿）= 清新/浪漫

## 三、值得尝试的新特效

### 1. 打开信封效果
- 点击后信封打开，露出里面的贺卡
- 用CSS transform + clip-path实现
- 可以做"抽出信件"的动画

### 2. 撕纸边缘效果
- 贺卡边缘做成撕纸质感
- CSS clip-path + SVG实现

### 3. 胶带/贴纸装饰
- 卡片角落加胶带、贴纸emoji装饰
- 营造手工感、真实感

### 4. 手写字打字机效果
- 祝福语一个字一个字出现
- CSS animation + steps() 实现

### 5. 折纸效果
- 卡片像折纸一样有立体感
- CSS 3D transform + perspective

### 6. 雪花/落叶覆盖
- 雪花/落叶飘落在卡片上
- 然后融化/消失的过渡效果

### 7. 霓虹灯效果
- 文字像霓虹灯一样发光闪烁
- text-shadow多层叠加

### 8. 星座连线效果
- 星星之间有连线，形成星座图案
- Canvas绘制动态连线

## 四、排版设计原则

### 文字层级
```
标题/名字（最大，最突出）
  ↓
日期/副标题（中等大小）
  ↓
祝福语（正常大小，易读）
  ↓
署名（小一些，靠右或居中）
```

### 留白
- 卡片内容不要太满
- 上下左右要有呼吸感
- 文字和边框保持足够距离

### 对齐
- 文字尽量居中对齐
- 祝福语如果较长可用两边对齐

## 五、动画设计原则

### 性能优先
- 优先用 `transform` 和 `opacity`
- 避免 `layout` 触发的动画（top/margin/width）
- 粒子多用 Canvas，少用 DOM

### 时间节奏
- 入场动画：0.5s - 1.5s
- 循环动画：3s - 8s
- 交互反馈：0.2s - 0.4s

### 错落感
- 同一页的元素，动画延迟错开
- 不要所有元素同时动

### cubic-bezier 常用曲线
```css
ease-out：正常减速（适合大多数）
ease-in-out：头尾慢（适合翻转）
cubic-bezier(0.34,1.56,0.64,1)：弹性（适合弹跳入场）
```
